# PorkHub

PorkHub is a **Laravel-based web app** for browsing and managing pork dishes. Clients can view dishes, see details, and add dishes for order.Admins are able to see Delivery Statistics, add Restaurant Branches, and can Add or Edit the dish menu.

## Features

- Browse dishes
- View dish details
- Add new recipes (admin)
- Add branches
- Manage Delivery
- Leave reviews 
- Responsive layout
